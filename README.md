## ✨ Features

- **🔧 Developer-Specific Nodes** - API, Database, Functions, and Bug nodes designed for developers
- **🔀 Visual Flow Mapping** - Drag and drop interface to visualize complex code logic and workflows
- **🔗 Shareable Links** - Generate and share your diagrams with team members
- **📥 Multiple Export Formats** - Export diagrams as PNG or JSON for documentation and version control
- **🌙 Dark Mode** - Built-in dark theme for comfortable extended use
- **🚀 Zero Login Required** - Start creating immediately—no authentication needed

## 📦 Installation

### Prerequisites

- Node.js 18+ (or latest LTS)
- npm or yarn package manager

### Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/devcanvas.git
   cd devcanvas
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173` (or the URL shown in terminal)

### Build for Production

```bash
npm run build
```

The optimized build will be generated in the `dist/` directory.

## 🚀 Usage

### Creating Your First Diagram

1. Click **"New Canvas"** to start a fresh diagram
2. Use the **Node Palette** on the left to add developer-specific nodes:
   - **API** - HTTP endpoints and services
   - **Database** - Data stores and queries
   - **Function** - Business logic and utilities
   - **Bug** - Issues and debugging points

3. **Connect nodes** by dragging from one node's port to another
4. **Customize** - Rename nodes and add descriptions
5. **Export** - Save as PNG for screenshots or JSON for version control

### Sharing Your Work

- Click the **"Share"** button to generate a unique link
- Share the link with teammates for collaboration and feedback
- No account required—links are publicly accessible

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Delete` | Remove selected node |
| `Ctrl+A` | Select all nodes |
| `Ctrl+C` | Copy selection |
| `Ctrl+V` | Paste selection |
| `Ctrl+Z` | Undo |
| `Ctrl+Shift+Z` | Redo |
